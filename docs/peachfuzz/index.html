🛡️ Defensive Fuzzing Capabilities

Find crashes before attackers do - safely in CI/CD pipelines

📊

PeachTrace Backend

Coverage-guided fuzzing with real execution traces

  • Code coverage instrumentation
  • Edge discovery and prioritization
  • Corpus minimization
  • Crash deduplication
🐍

Atheris Backend

Native Python fuzzing with libFuzzer engine

  • Structure-aware mutation
  • Fast execution (native speed)
  • Sanitizer integration (ASan, UBSan)
  • Industry-standard libFuzzer
⚙️

libFuzzer Integration

C/C++ fuzzing for performance-critical parsers

  • Clang instrumentation
  • Memory error detection
  • Continuous fuzzing mode
  • OSS-Fuzz compatible
🎯

Target Support

Fuzz all your parsers and APIs safely

  • JSON parsers (stdlib, orjson, rapidjson)
  • OpenAPI schema validators
  • GraphQL query parsers
  • Webhook handlers
  • XML/YAML parsers
  • Custom protocols
🔄

Crash Minimization

Automatically reduce crashing inputs to minimal reproducers

  • Delta debugging algorithm
  • Character-level minimization
  • Preserves crash trigger
  • 100-10000x size reduction typical
🧪

Regression Tests

Auto-generate pytest reproducers from crashes

  • One crash → one test function
  • Parametrized test generation
  • Immediate CI/CD integration
  • Version control friendly
