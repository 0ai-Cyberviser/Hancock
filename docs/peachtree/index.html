🎯 Enterprise-Grade Dataset Management

Transform raw security data into production-ready training datasets

✅

Quality Control

Automated validation and filtering of training data

  • Format validation (JSONL, JSON, CSV)
  • Schema enforcement
  • Confidence scoring
  • Outlier detection
🔄

Deduplication

MD5-based duplicate detection across datasets

  • MD5 hash fingerprinting
  • Cross-dataset dedup
  • Preserve unique samples only
  • ~30-40% reduction typical
📜

Provenance Tracking

Full audit trail for every training sample

  • Source attribution
  • Generation timestamp
  • Transformation history
  • Compliance-ready logs
📊

JSONL Export

Industry-standard format for LLM fine-tuning

  • Compatible with Unsloth, LoRA
  • Hugging Face Datasets ready
  • Streaming-friendly format
  • Compression support
🧠

Multi-Source Ingestion

Aggregate data from across your security stack

  • Hancock AI outputs
  • PeachTrace OSINT reports
  • PeachFuzz/CactusFuzz findings
  • Custom collectors
♻️

Recursive Self-Improvement

Datasets improve AI, which generates better datasets

  • Feedback loop with Hancock
  • Automatic quality metrics
  • Model performance tracking
  • Continuous optimization

🔄 Recursive Dataset Pipeline

From raw security data to production-ready training datasets

1

Ingest

Collect data from Hancock, PeachTrace, fuzzers, custom sources

2

Validate

Schema checks, format validation, confidence filtering

3

Deduplicate

MD5 hashing removes duplicates across all datasets

4

Export

Generate JSONL files ready for fine-tuning

5

Train

Fine-tune Hancock models (Mistral, Qwen, etc.)

6

Improve

Better models generate higher-quality data → Loop

⌨️ CLI Interface

Simple commands for powerful dataset operations

peachtree
$ peachtree ingest --source peachtrace_reports/ --type osint
✓ Ingested 87 OSINT reports
 
$ peachtree deduplicate --method md5
✓ Removed 34 duplicates (39.1% reduction)
Unique samples: 53
 
$ peachtree export --format jsonl --output hancock_osint_v2.jsonl
✓ Exported 53 samples
File: hancock_osint_v2.jsonl (247 KB)
 
$ peachtree stats
Total Datasets: 8
Total Samples: 3,402
Unique Samples: 2,187
Deduplication Rate: 35.7%
Avg Confidence: 0.87
 
🌳 Ready for fine-tuning with hancock_finetune.py
