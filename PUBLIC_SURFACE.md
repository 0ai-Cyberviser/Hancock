# Public Surface Map

This repository is part of the `0AI` / CyberViser portfolio operated by Johnny
Watters (`0ai-Cyberviser`).

## Canonical surfaces

- Portfolio hub: `https://0ai-cyberviser.github.io/0ai/`
- Source code, issues, and PRs: the GitHub repository that renders this file
- Repository docs: [`docs/`](docs/) and [`docs/openapi.yaml`](docs/openapi.yaml)
- Support routing: [`SUPPORT.md`](SUPPORT.md)
- Security reporting: [`SECURITY.md`](SECURITY.md)
- Ownership and branding: [`OWNERSHIP.md`](OWNERSHIP.md)

## Link policy

- Use the portfolio hub for company-level or cross-repository navigation.
- Use repo-relative links for repository-local docs whenever possible so forks
  stay portable.
- Avoid hard-coding temporary deployment hosts or stale microsite URLs in
  README files, templates, metadata, and docs.
- If a PR changes support, security, ownership, or public docs routing, update
  this file, the README, and the PR checklist together.

## Maintainer note

If a dedicated project site is republished later, treat that as an intentional
surface change and update every repo-facing reference in one pass rather than
mixing old and new destinations.
